Criptografia Clássica: Cifra de César

Como utilizar a Cifra de César?

Para criptografar uma mensagem utilizando a Cifra de César, precisamos determinar previamente uma chave de deslocamento . Essa chave de deslocamento é um número que indica a quantidade de casas que uma letra se deslocará no alfabeto, a partir do sentido horário. Após determinar esse número, trocamos cada letra da mensagem pela letra correspondente.

Por exemplo:
Número de deslocamento = 3
Mensagem = A LIGEIRA RAPOSA MARROM SALTOU SOBRE O CACHORRO CANSADO
Mensagem Cifrada = D OLJHLUD UDSRVD PDUURP VDOWRX VREUH R FDFKRUUR FDQVDGR
